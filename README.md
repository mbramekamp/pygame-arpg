# Python ARPG

This is a work in progress pygame arpg.

## Dependencies
- pygame-ce
- opensimplex

## Goal of the game
- Player should clear a map filled with enemies.
- Each enemy drops XP after its death
- XP can be picked up by a player
- once a xp threshold is over due, the player levels up
- player gains the chance to add/updgrade/remove one of his equipped skills.
